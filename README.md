# OneRoundSTL

OneRoundSTL: In-Database Seasonal-Trend Decomposition

- `code/` includes the codes for experiments
- `data/` includes the datasets for experiments

## Usage

Dev Environment:

- Java 8
- Maven

## Run

```
mvn clean package -DskipTests
cd target
java -jar tsdb-stl-1.0-SNAPSHOT.jar
```
